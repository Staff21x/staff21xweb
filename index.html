atencion_cliente.py
def procesar_consulta(mensaje, cliente_id):
    stock = consultar_inventario(mensaje.sku)
    if stock.disponible:
        responder_wsp(cliente_id, stock.info)
        ofrecer_turno(cliente_id)
    else:
        notificar_reposicion(cliente_id)
    registrar_bitacora(evento="consulta")
  • Hasta 1.000 mensajes WhatsApp / mes
  • Consulta stock y disponibilidad en tiempo real
  • Agenda turnos y citas desde el chat
  • Escala a humano cuando corresponde

def monitoreo_diario():
    vencidos = query_facturas(
        estado="vencido", dias=">0"
    )
    for f in vencidos:
        enviar_recordatorio(f.cliente)

TD–Compras
precios · cotizaciones · Excel · homologación

Analiza listas de precios, homologa productos entre proveedores y gestiona el proceso completo de cotización — incluyendo Excel estructurado y coordinación por correo.

compras.py
def homologar_proveedores(lista_a, lista_b):
    df = merge_by_id(lista_a, lista_b)
    df["precio_optimo"] = df.apply(
        lambda r: min(r.precio_a, r.precio_b)
    )
    exportar_excel(df, "comparativa.xlsx")
    enviar_cotizaciones(proveedores=["A","B"])
  • Análisis y comparación de listas de precios
  • Envío de solicitudes de cotización por correo
  • Homologación de IDs entre proveedores
  • Estructuración de datos en Excel
